# Car-Camping-Assist-Platform
  라즈베리파이와 GPS모듈, 가스센서등을 사용하여 차박(Car Camping)에 필요한 서비스를 제공하는 IoT Platform 입니다.
  

# Used Hardware
  Raspberry Pi 3        / 4EA
  Raspberry Pi Monitor  / 2EA
  5V LED Strip          / 2EA
  GAS Sensor MQ-2       / 2EA
  Buzzer                / 2EA
  GPS Module NEO-6M     / 2EA
  PIR Sensor HC-SR501   / 2EA
